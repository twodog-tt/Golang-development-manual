    口述模板（每题 3 分钟）¶

    1. 业务背景：你在项目中负责哪块、解决什么问题
    2. 架构决策：为什么 WebSocket 优先、为什么 MQ 拆链路、为什么 N 确认
    3. 故障案例：reorg 回滚 / 提现重试 / RPC 抖动 — 怎么发现、怎么修
    4. 指标：lag、P99、连接数、对账差异率
